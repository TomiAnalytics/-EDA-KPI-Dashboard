# EDA & Storytelling: Business KPI Dashboard

## Objective
Analyze key business metrics (sales, revenue, customer behavior) and create a KPI dashboard for business decision-making.

## Tools & Technologies
Python (Pandas, Matplotlib, Seaborn), Power BI, Excel, Jupyter Notebook

## Methodology
1. Data cleaning and preprocessing  
2. Exploratory Data Analysis (EDA)  
3. KPI calculation and trend analysis  
4. Dashboard design in Power BI  
5. Storytelling through visuals to highlight insights

## Key Insights
- Total revenue trends and seasonality  
- Top-performing product categories  
- Customer purchase behavior patterns

## Recommendations
- Adjust inventory for top-selling products  
- Focus marketing campaigns on low-engagement customer segments
